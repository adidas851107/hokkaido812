# 北海道親子旅行助手

## 部署
1. 解壓縮。
2. 將資料夾內所有檔案上傳到 `adidas851107/hokkaido-trip`。
3. Vercel 選 Add New → Project → Import 該 Repository → Deploy。

## 本機預覽
```bash
npm install
npm run dev
```

## 已包含
- 全中文手機介面
- 7天完整行程與雨天備案
- Google Maps導航與QR Code
- 每日油資、高速、停車估算
- 寶寶紀錄、旅費紀錄、行李清單
- PWA主畫面與基礎離線快取

航班時間、費用與煙火仍應以出發前官方公告為準。
